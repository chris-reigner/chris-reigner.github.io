# AI Agents Engineering

A practitioner's handbook for designing, building, and operating production AI agent systems.

---

| Section | What it covers |
|---|---|
| **Foundations** | Vocabulary, concepts, and taxonomy — what agents are and how to classify them |
| **Architecture & Design** | System shapes, hierarchy design, interaction protocols, task decomposition |
| **Runtime Engineering** | Memory, context management, skills, and human-in-the-loop |
| **Agent Ops** | Release lifecycle, artifact management, prompt lifecycle, and security vulnerabilities |
| **Evaluation** | How to measure agent quality — offline, online, and continuously |
