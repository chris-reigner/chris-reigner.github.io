# AI Engineering Handbook

Welcome to the AI Engineering Handbook - your comprehensive guide to becoming a Full Stack ML Engineer.

This repository aims at giving any reader the ability to become a Full Stack ML engineer across the wild ecosystem of Data and Computer Science. Some parts focus on Natural Language Processing that became LLM driven in recent years, with a specific focus on MLOps and LLMOps.

## What You'll Learn

- **ML Engineering**: Core engineering concepts, async programming, Docker, and modern tools
- **GenAI Engineering**: Deep learning, LLM pipelines, fine-tuning, RAG, evaluation, and LLMOps
- **AI Agents Engineering**: Building intelligent agents and multi-agent systems
